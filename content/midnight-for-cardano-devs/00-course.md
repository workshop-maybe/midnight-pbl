# Midnight for Cardano Developers

## Course Description

You know Cardano. You build on it. Now Midnight is launching as a partner chain with privacy features Cardano doesn't have. This course teaches you to build on Midnight by connecting every concept to what you already know — EUTXO, datums, validators, native assets, transaction building.

By the end, you won't just know Midnight. You'll know when to use it, when Cardano is the better choice, and how to architect systems that use both.

The course is a living document. Your assignments aren't quizzes — they're contributions that make the course better. Your credential proves you engaged deeply enough to improve what came before you.

## Module Overview

| Module | Title | Focus |
|--------|-------|-------|
| 101 | Your First Midnight DApp | Hands-on from minute one. Deploy a contract, feel the proof generation, see the dual state. |
| 102 | Midnight Architecture Through Cardano Eyes | Map what you know (EUTXO, datums, validators) to Midnight's model (dual ledger, Kachina, Compact). |
| 103 | Compact for Cardano Developers | The language itself. Private/public annotations, circuits, witnesses — connected to patterns you recognize. |
| 104 | The Privacy Model in Practice | Build something that uses privacy. Experience the cost, the flow, the tradeoffs. |
| 105 | Token Economics and Application Design | NIGHT/DUST, the token matrix, sponsored transactions, deployment considerations. |
| 106 | When Midnight, When Cardano, When Both? | The capstone question. Design an architecture that uses both chains for what each does best. |
